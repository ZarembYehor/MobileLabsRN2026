# Лабораторна робота №3: Використання кастомних жестів у React Native та стилізація інтерфейсу мобільного застосунку.

## Мета
Навчитися працювати з жестами користувача у мобільному застосунку, реалізувати взаємодію через різні типи жестів та застосувати сучасні підходи стилізації у React Native.

## Опис реалізованого функціоналу

### 1. Інтерактивні жести
Реалізовано комплексний ігровий елемент, який реагує на широкий спектр дій:
- **Taps & Long Press:** Одинарний тап, подвійний тап та утримання протягом 3 секунд для виконання спеціального завдання.
- **Pan Gesture:** Можливість вільного переміщення об'єкта по екрану з поверненням у вихідну точку за допомогою пружинної анімації.
- **Fling:** Швидкі свайпи вліво та вправо для нарахування випадкової кількості бонусних балів.
- **Pinch & Rotation:** Одночасна зміна масштабу та обертання об'єкта двома пальцями.
- **Оптимізація:** Використано `runOnJS` для безпечної взаємодії між потоком жестів та логікою додатка.

### 2. Глобальний стан
Для керування прогресом користувача впроваджено легковажне сховище:
- **Quest Tracking:** Відстеження виконання списку з 9 завдань.
- **Live Updates:** Миттєве оновлення рахунку та статусу завдань на всіх екранах без зайвих ререндерів.

### 3. Динамічна стилізація
Додаток підтримує повноцінну світлу та темну теми:
- **Context API + Styled Provider:** Глобальний контекст передає об'єкт теми в `ThemeProvider` бібліотеки `styled-components`.
- **Custom Components:** Всі UI-елементи переписані на стилізовані компоненти для чистоти коду.
- **Адаптивність:** Кольори іконок, фонів та рамок автоматично змінюються при перемиканні в налаштуваннях.

## Інструкція із запуску

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/ВАШ_ЛОГІН/MobileLabsRN2026.git
2. **Перейдіть до папки проєкту**
   ```bash
   cd MobileLabsRN2026/lab3
3. **Встановіть необхідні залежності:**
   ```bash
   npm install
4. **Запустіть Expo CLI:**
   ```bash
   npx expo start
5. **Відкрийте додаток:**
   * Скануйте QR-код через додаток Expo Go на вашому смартфоні.
   * Або натисніть **a** для запуску на Android Emulator.
   * Або натисніть **i** для запуску на iOS Simulator.

## Висновки

У результаті виконання лабораторної роботи було успішно опановано стек технологій для створення високоінтерактивних мобільних інтерфейсів у React Native, зокрема поєднання складних багатопотокових жестів та анімацій. Використання бібліотек `react-native-gesture-handler` та `react-native-reanimated` дозволило реалізувати плавне керування об'єктами через свайпи, тапи, обертання та масштабування, враховуючи специфіку розподілу навантаження між UI та JS потоками. Завдяки впровадженню `Zustand` було досягнуто ефективного керування глобальним станом прогресу користувача без перевантаження дерева компонентів, а використання `Styled Components` забезпечило гнучку реалізацію адаптивної системи тем, що в сукупності дозволило створити продуктивний, масштабований та архітектурно правильний застосунок із сучасним підходом до стилізації та обробки користувацького досвіду.
